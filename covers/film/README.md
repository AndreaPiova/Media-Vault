# Cover Film

Poster e backdrop dei film.

- Poster: `film_[slug]_[anno].jpg`
- Backdrop: `film_[slug]_[anno]_backdrop.jpg`

Il campo "slug" è il titolo (in italiano se il film è noto con quel titolo nell'app, altrimenti in lingua originale) in minuscolo con underscore al posto degli spazi (es. "Ricomincio da capo" -> `ricomincio_da_capo`).

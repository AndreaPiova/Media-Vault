# Cover Variant / Artbook Manga

Copertine di variant, edizioni speciali e artbook manga.

- Variant/edizione speciale: `variant_[manga]_[nome variante]_vol[N].jpg`
- Artbook: `artbook_[manga]_[nome artbook].jpg`

Il campo "slug" è in minuscolo con underscore al posto degli spazi (es. "Berserk Variant A vol.41" -> `variant_berserk_variant_a_vol41.jpg`).
